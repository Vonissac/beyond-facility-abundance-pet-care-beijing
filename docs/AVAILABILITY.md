# Data and code availability

This repository contains the derived data and code required to verify the reported aggregate analyses, manuscript tables, supplementary tables, validation boundaries and model-explanation figure. The raw source records are not redistributed when governed by original access terms, licensing restrictions or privacy-sensitive provider conditions. The released analysis-ready grid table records the derived fields used by the reproducibility scripts.

The repository URL may be cited in the article data-availability statement as the public location of the released data, code, reference outputs and figure files.
